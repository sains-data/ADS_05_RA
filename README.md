----------------------------------------------------------
Penjelasan Pengaruh Aktivitas Organisasi dan Waktu Belajar
-----------------------------------------------------------  

Berdasarkan analisis data mahasiswa dengan variabel keterlibatan organisasi (aktif/tidak aktif) dan waktu belajar per minggu, hasil menunjukkan bahwa aktivitas organisasi tidak memiliki pengaruh signifikan 
terhadap waktu belajar.

-----------------------
Cara Menjalankan Script
-----------------------

Script CodeR_05_RA.R dijalankan dengan menyiapkan dataset Dataset ads clean kelompok 5.xlsx di folder data, menginstal paket readxl, dplyr, ggplot2, car, dan effsize, lalu menjalankan script dari awal di RStudio. Script ini memuat data, membersihkan kolom, membuat kategori organisasi, menghasilkan statistik deskriptif, membuat visualisasi, serta melakukan uji normalitas, uji Levene, uji t dua sampel, dan menghitung effect size.

----------------------
Paket R Yang Digunakan
----------------------

Dataset berisi dua variabel utama: Jam_Belajar (numerik, variabel dependen) dan Keterlibatan Organisasi (kategori: Aktif/Tidak Aktif, sebagai variabel independen). Metode analisis mencakup pembersihan data, visualisasi (boxplot & histogram), statistik deskriptif, serta pengujian inferensial.
